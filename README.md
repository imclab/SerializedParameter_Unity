SerializedParameter_Unity
=========================

エディタ上でゲーム実行時に変更したフィールド変数を保持します。

![pre game](01.jpg)

![playing game](02.jpg)  
起動後に値を変更

![exit game](03.jpg)  
プレイ終了後も値を保持

#使い方

publicの変数に[SerializeParam]を付ける。  
構造体・クラス・プリミティブな値どれでも可…だと思う。
